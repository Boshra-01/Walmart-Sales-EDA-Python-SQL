# Walmart Sales Data Analysis with Python and SQL

**Dataset Link**: [Walmart Sales Dataset](https://www.kaggle.com/datasets/najir0123/walmart-10k-sales-datasets)

---

## Project Overview

This project is a comprehensive data analysis pipeline built to derive actionable business insights from Walmart sales data. It leverages **Python** for data processing and visualization, **SQL** for advanced querying and structured problem-solving methodologies to address key business challenges.  

---

## Results and Insights

The analysis revealed top-performing product categories, branches with the highest sales, commonly used payment methods etc. It also provided insights into the most profitable product categories and locations, along with trends in customer ratings, payment preferences and peak shopping hours.
